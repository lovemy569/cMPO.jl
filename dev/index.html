cMPO.chi — Method

cMPO.chi2 — Method

Calculate the dynamic susceptibility χ''(ω) = - (π/Z) Σ{m,n} Ṡ{nm}ᶻ Ṡ{mn}ᶻ (e^{-βΛₘ} - e^{-βΛₙ}) δ(ω - Λₘ + Λₙ), where δ(x) = lim{η→0} (1/π) (η / (x² + η²)).
